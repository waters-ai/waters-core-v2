## Вывод
Оба узла демонстрируют работоспособность своих P2P возможностей:
- mDNS для локального обнаружения
- TCP sync для прямого узел-узел соединения
- Система каналов для обмена сообщениями

Для соединения через интернет (между IP 171.22.180.238 и локальным узлом) потребуется настройка статических пиров, так как mDNS не работает через интернет.
