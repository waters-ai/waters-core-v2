## Протоколы обмена

### Между агентами миссии (Redis pub/sub)

```
Канал: mission:state
Формат: JSON
Публикует: CoordinatorAgent / любой агент
Тело: {
  "type": "state_update",
  "agent": "collector.1",
  "state": "collecting|analyzing|idle|error",
  "payload": {}
}
```

```
Канал: mission:findings
Формат: JSON
Публикует: AnalyzerAgent / PatternMatcherAgent
Тело: {
  "type": "new_finding",
  "finding_id": "uuid",
  "source": "nasa_fireball_20260515",
  "confidence": 0.87,
  "data": { ... }
}
```

### Между миссией и центром (HTTP bridge → Kafka)

```
Миссия → Центр (findings):
  POST https://238.bridge.waters/mission/1/findings
  Body: { findings: [...], heartbeat: {...} }

Центр → Миссия (orders):
  GET https://238.bridge.waters/mission/1/orders
  Response: { orders: [...], timestamp: "..." }
```

## Уровни автономии

| Уровень | Описание | Условие |
|---------|----------|---------|
| **L0** | Полная мощность | Солнце + Sat-линк + DeepSeek API | Все агенты, полный анализ |
| **L1** | Экономия API | Солнце есть, Sat нет | Ollama локально, queue для findings |
| **L2** | Автономный | Солнце есть, Sat нет, Ollama | Полная миссия без центра. Sync при восстановлении Sat |
| **L3** | Энергосбережение | Облачность, мало солнца | Только критичные агенты, дроны на зарядку |
| **L4** | Safe mode | Ночь / нет солнца | Только приём данных, логирование. Дроны на базе |
