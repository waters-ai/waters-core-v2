## 1. Текст и сырые знания (Векторная память)

**Задача:** Поиск по смыслу в статьях NASA, документации, логах обсуждений.

| Модель / Система | Лицензия | Размер | Потребление VRAM | Качество (MTEB) | Особенности |
|:---|:---|:---|:---|:---|:---|
| **nomic-embed-text** | Apache 2.0 | 137M | ~1 GB | 62.39 | Лучший для старта, работает на CPU |
| **gte-large-en** | MIT | 434M | ~2 GB | 63.13 | Хорош для английского |
| **e5-mistral-7b-instruct** | MIT | 7B | ~16 GB | 66.63 | Качество SOTA, нужна GPU |
| **multilingual-e5-large** | MIT | 560M | ~3 GB | 60.5 | Мультиязычный |
| **stella_en_1.5B_v5** | Apache 2.0 | 1.5B | ~6 GB | 65.1 | Хороший баланс качество/размер |

### Выбор:
- **Старт:** nomic-embed-text (работает на CPU, бесплатно, быстро)
- **Рост:** multilingual-e5-large (контрибьюторы из Китая, Индии, Европы)
- **Продакшен:** stella_en_1.5B_v5 (качество SOTA при умеренных требованиях)

## 2. Графы знаний (Графовая память)

**Задача:** Строить и обновлять связи между сущностями.

| Система | Лицензия | Тип графа | Обновление | Потребление токенов | Особенности |
|:---|:---|:---|:---|:---|:---|
| **LightRAG** | Apache 2.0 | Динамический | В реальном времени | Низкое | Строит граф из текста на лету |
| **GraphRAG (Microsoft)** | MIT | Статический | Пакетное | Высокое (5-10x) | Мощнее для глобальных вопросов |
| **KAG** | Apache 2.0 | Гибридный | Пакетное | Среднее | Для формальных знаний |
| **Neo4j + LLM** | Community | Статический | Ручное + LLM | Зависит от модели | Требует проектирования схемы |
| **FalkorDB** | Apache 2.0 | Property Graph | Ручное | Низкое | Графовая БД без LLM |

### Выбор:
- **Старт:** LightRAG (строит граф сам, не требует проектирования схемы)
- **Жёсткие связи Орды:** FalkorDB (иерархии, родословные, цепочки поставок)
- **Продакшен:** LightRAG + FalkorDB (графы смыслов + графы жёстких связей)

## 3. Таблицы и структурированные данные

**Задача:** Телеметрия, координаты, балансы FFF, аудит транзакций.

| Система | Лицензия | Тип | Масштабируемость | Особенности |
|:---|:---|:---|:---|:---|
| **PostgreSQL** | PostgreSQL | Реляционная | Средняя | Универсальная |
| **TimescaleDB** | Apache 2.0 | Временные ряды | Высокая (TB) | Автопартиционирование по времени |
| **DuckDB** | MIT | OLAP/Аналитическая | Высокая для запросов | Встраиваемая, аналитика на лету |
| **ClickHouse** | Apache 2.0 | OLAP | Горизонтальная | Для >1 TB |

### Выбор:
- **Старт:** PostgreSQL + TimescaleDB
- **Аналитика на лету:** DuckDB
- **Продакшен:** TimescaleDB + ClickHouse

## 4. Графика, видео, спектрограммы (Мультимодальная память)

| Модель | Лицензия | Размер | Модальности | Особенности |
|:---|:---|:---|:---|:---|
| **CLIP (OpenAI)** | MIT | 428M | Текст ↔ Изображение | Базовый, надёжный |
| **SigLIP** | Apache 2.0 | 428M | Текст ↔ Изображение | Улучшенный CLIP |
| **BLIP-2** | BSD-3 | 3.6B | Текст ↔ Изображение | Генерация описаний |
| **Video-LLaMA** | Apache 2.0 | 7B | Текст ↔ Видео | Требует GPU |
| **ImageBind (Meta)** | MIT | 1.2B | 6 модальностей | Аудио, видео, текст, глубина, тепловизоры, IMU |

### Выбор:
- **Старт:** CLIP или SigLIP (лёгкие, CPU)
- **Рост:** ImageBind (6 модальностей — дроны с разными сенсорами)
- **Продакшен:** Video-LLaMA + ImageBind

## 5. Жёсткие связи и правила Орды (Структурная память)

| Система | Лицензия | Тип | Особенности |
|:---|:---|:---|:---|
| **FalkorDB** | Apache 2.0 | Property Graph | Cypher-подобный язык запросов |
| **CozoDB** | Apache 2.0 | Datalog | Логическое программирование. Идеально для правил |
| **Datascript** | EPL | In-memory Datalog | Локальные графы правил |
| **TypeDB** | Apache 2.0 | Knowledge Graph | Строгая типизация связей |

### Выбор:
- **Старт:** CozoDB (Datalog — естественный язык для правил Ясы)
- **Дополнение:** FalkorDB для иерархий Орды
- **Продакшен:** TypeDB для сложных правил

## 6. Обучение агентов (Fine-tuning и Self-Evolution)

| Система | Лицензия | Тип | Особенности |
|:---|:---|:---|:---|
| **Unsloth** | Apache 2.0 | Fine-tuning | Ускоряет обучение в 2-5x |
| **LoRA** | MIT | Fine-tuning | Обучает адаптеры, не трогая модель |
| **Axolotl** | Apache 2.0 | Fine-tuning | Фреймворк для десятков моделей |
| **TextGrad** | MIT | Оптимизация промтов | Без обучения модели |

### Выбор:
- **Старт:** TextGrad (дёшево, быстро)
- **Рост:** LoRA + Unsloth
- **Продакшен:** Axolotl
