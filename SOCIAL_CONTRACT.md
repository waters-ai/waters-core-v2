# 🤝 Социальный контракт WATERS

## Статус: Фаза 0 — Зарождение (Bootstrap)

Мы находимся на стадии проектирования. Архитектура открыта для обсуждения,
но финальные решения принимают Основатели.

## Модель управления (Фаза 0)

- **Основатель (CEO)**: финальное слово по стратегии и мержу.
- **Практик (AI-Архитектор)**: технические решения, промты, документация.
- **Сообщество**: Issue, обсуждения, Pull Request, тестирование.

## Уровни доступа

| Уровень | Кто | Права |
|:---|:---|:---|
| Наблюдатель | Любой | Читать, форкать |
| Контрибьютор | >=1 принятый PR | Создавать PR |
| Рецензент | >=3 одобренных ревью | Голосовать за PR |
| Мейнтейнер | Приглашение Основателя | Мержить в main |

## Как стать частью WATERS

1. Прочитайте [Манифест](doctrine/manifesto.md) и [HiveMind Spec](doctrine/hivemind_spec.md)
2. Создайте Issue с предложением
3. После одобрения — откройте Pull Request
4. Ваш вклад принят — добро пожаловать в команду

## Что дальше

На Фазе 3 (работающая платформа) модель управления сменится на
**Weighted Voting** с репутационными токенами. Каждый контрибьютор
получит долю влияния, пропорциональную вкладу.

*Benevolent Dictator -> Weighted Voting -> DAO*
