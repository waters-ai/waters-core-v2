## 1. Цель миссии / Mission Objective / 任务目标

**Основная:** Отработка технологий поиска метеоритов и подземных вод на Земле для последующей адаптации к лунной миссии Deima.

**Primary:** Test meteorite and groundwater search technologies on Earth for subsequent adaptation to the Deima lunar mission.

**主要目标：** 在地球上测试陨石和地下水搜索技术，以便后续适应 Deima 月球任务。

### Критерии успеха / Success Criteria / 成功标准

| Критерий / Criterion / 标准 | Цель / Target / 目标 | Измерение / Measurement / 测量 |
|:---|:---|:---|
| Найден метеорит | > 100 г | Взвешивание, спектральный анализ |
| Обнаружена вода в грунте | > 1% концентрация | Влагомер, LIBS, спектрометр |
| Точность карты | < 1 м | RTK GPS |
| Полнота телеметрии | > 95% | Kafka offset |
| Все дроны возвращены | 100% | Телеметрия |

### 2.1 Агенты / Agents / 智能体

| Агент / Agent / 智能体 | Код / Code / 代码 | Роль / Role / 角色 | Навык / Skill / 技能 |
|:---|:---|:---|:---|

### 2.2 Дроны / Drones / 无人机

| Параметр / Parameter / 参数 | Значение / Value / 值 |
|:---|:---|
| **Тип** | Аналог DJI Matrice 350 RTK + Mavic 3M |

| **Бортовой накопитель** | 256 GB (буферизация при потере связи) |

## 3. Потоки данных / Data Flows / 数据流

```
Дрон ──▶ Kafka (telemetry.raw.v1) ──▶ TimescaleDB ──▶ Аналитика ──▶ Neo4j
  │                                                                   │
  └──▶ Бортовой накопитель ──▶ (при восстановлении) ──▶ Delta sync ──┘
```

| Шаг / Step / 步骤 | Откуда / From / 来源 | Куда / To / 目标 | Формат / Format / 格式 |
|:---|:---|:---|:---|
| 1 | Сенсоры дрона | Kafka `telemetry.raw.v1` | Avro (сериализованные фреймы) |
| 2 | Kafka | TimescaleDB | JSON (гипертаблицы) |
| 3 | TimescaleDB | Neo4j | Граф (состояния, связи, аномалии) |
| 4 | Neo4j | LightRAG | Граф знаний (причины, следствия) |
| 5 | LightRAG | Агенты (MCP memory_graph_query) | Текст (ответы на запросы) |

## 7. Технологическая демонстрация / Tech Demo / 技术演示

Данная миссия проверяет следующие технологии для переноса на Луну:

| Технология / Technology / 技术 | Земля / Earth / 地球 | Луна / Moon / 月球 | Разница / Delta / 差异 |

| LIBS спектрометр | Атмосферное давление | Вакуум | Калибровка |
| Буровой инструмент | 1 atm, +20°C | Vacuum, -180°C | Термозащита |
| Автономный полёт | 4G/5G покрытие | Прямая видимость | DTN-протокол |
