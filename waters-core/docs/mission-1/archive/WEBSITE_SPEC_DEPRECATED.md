# Сайт виртуальной миссии — Поиск метеоритов

**Статус:** Проектирование
**Дата:** 2026-05-15
**Автор:** Архитектор v1.0

---

## 1. Назначение

Публичный сайт, который визуализирует ход виртуальной миссии по поиску метеоритов.
Любой желающий может:
- Наблюдать за работой агентов в реальном времени
- Смотреть найденные метеориты и треки
- Изучать данные NASA и спектры
- Видеть статистику миссии

## 4. Tech Stack

### 4.1. Frontend

| Компонент | Технология | Обоснование |
|-----------|-----------|-------------|
| **Фреймворк** | React 19 + TypeScript | Стандарт, богатая экосистема |
| **Сборка** | Vite | Быстрая, современная |
| **3D Globe** | Three.js + Globe.GL | Визуализация треков на глобусе |
| **Графики** | D3.js | Спектры, статистика |
| **WebSocket** | socket.io-client | Реал-тайм лента находок |
| **CSS** | Tailwind 4 | Быстрая стилизация |

### 4.2. Backend

| Компонент | Технология | Обоснование |
|-----------|-----------|-------------|
| **Фреймворк** | FastAPI (Python) | Асинхронный, генерация OpenAPI |
| **WS** | FastAPI WebSocket | Реал-тайм события |
| **DB** | SQLite (прототип) → PostgreSQL | Простота старта |
| **Bridge** | HTTPX + aioredis | Чтение из mission-control |

### 4.3. Деплой

```yaml
# docker-compose.yml для сайта
services:
  backend:
    build: ./backend
    ports: ["8000:8000"]
    env:
      MISSION_API: "http://mission-control:8080/api/v1"
      REDIS_URL: "redis://redis:6379"

  frontend:
    build: ./frontend
    ports: ["80:80"]
    depends_on: [backend]

  redis:
    image: redis:7-alpine
```

## 5. API endpoint'ы

| Метод | Путь | Назначение |
|-------|------|------------|
| `GET` | `/api/v1/mission/status` | Статус миссии |
| `GET` | `/api/v1/findings?page=1&limit=20&type=meteorite` | Список находок |
| `GET` | `/api/v1/findings/{id}` | Детали находки |
| `GET` | `/api/v1/agents` | Список агентов |
| `GET` | `/api/v1/agents/{id}` | Детали агента |
| `GET` | `/api/v1/stats` | Статистика миссии |
| `WS` | `/ws/events` | Реал-тайм события (находки, статусы) |
