**Статус:** Черновик Системного Архитектора. Фаза 0.

**Диаграмма:** [bionic_brain.svg](diagrams/bionic_brain.svg)

## Концепция

Платформа WATERS строится не как монолит и не как набор микросервисов,
а как **живой организм**, состоящий из специализированных клеток,
объединяющихся в органы и системы.

Клетки эволюционируют: самые эффективные цепочки "пожираются"
платформой и превращаются в автономных агентов ("пирожков").

---

## Три отдела мозга платформы

1. **Правое полушарие** — Интуиция, ассоциативный поиск.
2. **Лимбическая система** — Память, эмоции, рефлексы.
3. **Ствол мозга** — Жизнеобеспечение, базовые функции.

## 1. Ствол мозга (Жизнеобеспечение)

Самые примитивные, но критически важные клетки.
Написаны на Rust/Go. Работают как вегетативная нервная система:
дышат, качают кровь, не спрашивая разрешения.

### Клетки ствола:

| Клетка | Функция | Технология | Аналог в организме |
|:---|:---|:---|:---|
| **respirer** | Мониторинг CPU/RAM/Disk, алерты | Prometheus + Rust | Дыхательный центр |
| **pumper** | Перекачка сообщений через Redpanda | Redpanda + Go | Сердце |
| **janitor** | Очистка логов, архивация в TimescaleDB | Rust + SQL | Почки/печень |
| **guardian** | Блокировка опасных команд (damage-control) | OpenCode plugin | Иммунная система |
| **echo** | Пинг агентов, измерение DTN-задержки | Rust + ICMP | Эхолокация |

### Принцип работы:
- Клетки ствола не принимают решений. Они выполняют базовые функции.
- При отказе любой клетки — алерт в лимбическую систему (cortisol).
- Клетки ствола не эволюционируют. Они стабильны как фундамент.

## 2. Лимбическая система (Память и эмоции)

Клетки быстрого реагирования. В основном на Python.
Это "гормоны" и "рефлексы" платформы.

### Клетки лимбической системы:

| Клетка | Функция | Технология | Аналог в организме |
|:---|:---|:---|:---|
| **cortisol** | Фиксация стресса: рост задержек, нехватка FFF | Python + Redis | Кортизол (гормон стресса) |
| **dopamine** | Начисление репутации и FFF за успех | Python + Solana | Дофамин (награда) |
| **oxytocin** | Скрепление контрактов между агентами | Python + HiveMind | Окситоцин (доверие) |
| **memory_b** | Быстрая память: текущее состояние агентов | Redis | Кратковременная память |
| **memory_t** | Глубинная память: архив всех событий | Redpanda/Kafka | Долговременная память |

### Принцип работы:
- **cortisol** срабатывает при задержке > порога → активирует режим экономии FFF.
- **dopamine** начисляет награду при успешном завершении миссии.
- **oxytocin** фиксирует контракт между агентами → запись в HiveMind Layer II.
- **memory_b** хранит состояние "прямо сейчас" (где агент, его баланс).
- **memory_t** хранит всю историю (каждое действие, каждый платёж).

## 3. Правое полушарие (Интуиция и ассоциативный поиск)

"Библиотекари" и "следопыты", которые роются в данных
и находят неочевидные связи.

### Клетки правого полушария:

| Клетка | Функция | Технология | Аналог в организме |
|:---|:---|:---|:---|
| **retriever_semantic** | Быстрый поиск по смыслу | ChromaDB + nomic-embed | Семантическая память |
| **retriever_graph** | Поиск связей в графе (иерархии, Яса) | FalkorDB / CozoDB | Структурная память |
| **retriever_multimodal** | Поиск похожих снимков, спектрограмм | CLIP / SigLIP | Зрительная кора |
| **daydreamer** | Поиск аномалий, построение гипотез | LightRAG + DeepSeek | Воображение |
| **muser** | Анализ старых решений, предложение шаблонов | LightRAG + TextGrad | Интуиция |

### Принцип работы:
- **retriever_semantic** ищет "похожие статьи" или "похожие логи".
- **retriever_graph** ищет жёсткие связи: "кто кому подчиняется", "какой ресурс где".
- **daydreamer** строит гипотезы: "аномалия в секторе G-23 может быть льдом".
- **muser** предлагает: "в прошлый раз в такой ситуации мы делали так".

## 4. Левое полушарие (Логика и быстрые решения)

| Клетка | Функция | Технология |
|:---|:---|:---|
| **validator** | Проверка JSON-схем, валидация PR | DeepSeek V4-Flash |
| **router** | Маршрутизация запросов между компонентами | MCP-шина (JSON-RPC) |
| **classifier** | Классификация задач по типу | DeepSeek V4-Flash |

---

## 5. Мозолистое тело (Координация)

**MCP-шина (JSON-RPC 2.0)** — единый протокол связи между всеми отделами.
Аналог USB-C для AI-агентов. Обеспечивает подключение внешних агентов.

---

## 6. Эволюция: Пожирание и специализация

Самые эффективные цепочки клеток не остаются просто набором скриптов.
Платформа отслеживает их через Kafka-лог и, если паттерн повторяется,
запускает механизм "сборки" в нового автономного агента.

### Процесс эволюции:


### Примеры эволюции:

**Агент-Шахтёр (Miner):**
- Паттерн: `retriever_multimodal` → `retriever_graph` → `dopamine`
- Смысл: Найти воду по спектрограмме → проверить по карте → получить награду
- Результат: Платформа пакует цепочку в агента "Miner" и регистрирует его

**Супервизор Роя (Overseer):**
- Паттерн: `cortisol` → `daydreamer` → `oxytocin` → `pumper`
- Смысл: Зафиксировать угрозу → построить план → заключить контракты → разослать приказы
- Результат: Платформа создаёт агента "Overseer" с повышенным рангом в Орде

## 7. Будущие органы (по мере роста)

| Орган | Из каких клеток | Функция |
|:---|:---|:---|-
| **Гипоталамус** | cortisol + dopamine + oxytocin | Центральный регулятор экономики FFF |
| **Гиппокамп** | memory_b + memory_t + muser | Центр обучения на прошлом опыте |
| **Мозжечок** | echo + pumper + guardian | Координация движения агентов в пространстве |
| **Префронтальная кора** | daydreamer + retriever_graph + validator | Стратегическое планирование миссий |
