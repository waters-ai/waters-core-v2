## Назначение

Telegram-бот для приёма задач от CEO напрямую. Позволяет отправить поисковый запрос без Kafka.

## Входные данные

| Сообщение | Формат |
|-----------|--------|
| `search: <запрос>` | Поиск информации по запросу |

## Выходные данные

| Тип | Куда |
|-----|------|
| Подтверждение задачи | Ответ в Telegram |
| `file_ready` | `planners.answers.v1` |

## Алгоритм

1. CEO пишет `search: <запрос>` в Telegram
2. TelegramBot создаёт Task с task_id, query, requester=telegram:{uid}
3. Task уходит в _process_task (поиск → валидация → отправка)
4. CEO получает подтверждение `Задача принята: {task_id}`
5. Результаты: file_ready + task_summary в Kafka

## Безопасность

- API-токен Telegram-бота
- Белый список пользователей (опционально)
- Если пользователь не в списке — `Доступ запрещён`
