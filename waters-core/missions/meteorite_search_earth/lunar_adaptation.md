| **Инфракрасный спектрометр** | Минералогия | SWIR (аналогично) | Калибровка под вакуум |

## 3. Адаптация протоколов HiveMind / HiveMind Protocol Adaptation / HiveMind 协议适配

### 3.1 Задержка связи / Communication Delay / 通信延迟

| Модель HiveMind / Model / 模型 | Земля / Earth / 地球 | Луна / Moon / 月球 | Адаптация / Adaptation / 适配 |
|:---|:---|:---|:---|
| **Военная** | 50 ms → мгновенный приказ | 2.5 s → пакетный приказ | Пакетная отправка команд, квитирование через DTN |
| **Корпоративная** | 50 ms → торги в реальном времени | 2.5 s → асинхронные торги | Тендер с таймаутом 30 с, результаты через DTN |
| **Научная** | 50 ms → диалог | 2.5 s → асинхронный диалог | Гипотеза → сбор данных → обсуждение через DTN |
| **Демократическая** | 50 ms → голосование | 2.5 s → заочное голосование | Бюллетени через Kafka, подсчёт через DTN |
| **Монархическая** | 50 ms → совет | 2.5 s → декрет | Декрет Монарха → исполнение без обсуждения |

### 3.2 DTN-профили / DTN Profiles / DTN 配置

| Профиль / Profile / 配置 | Задержка / Delay / 延迟 | Пропускная способность / Bandwidth / 带宽 | Применение / Usage / 应用 |
|:---|:---|:---|:---|
| `lunar_delay_2.5s` | 2.5 s RTT | 1-10 Mbps | Лунная орбита/поверхность |
| `lunar_surface_direct` | 0.1 s | 100 Mbps | Прямая видимость (10 km) |
| `lunar_farside_relay` | 5-10 s | 100 Kbps | Через ретранслятор (L2) |
| `earth_mars_min` | 5 min | 1-10 Mbps | В оппозиции |
| `earth_mars_max` | 20 min | 100 Kbps | В соединении |

### 3.3 Автономное принятие решений / Autonomous Decision Making / 自主决策

При потере связи с Землёй:

1. **Tier 1 (0-30 с)**: дрон продолжает текущую задачу, буферизует данные
2. **Tier 2 (30-300 с)**: дрон завершает задачу, переходит в режим ожидания команд
3. **Tier 3 (300+ с)**: дрон активирует аварийный протокол — возврат на базу

На Луне все таймауты увеличены в 50× из-за задержки связи.
