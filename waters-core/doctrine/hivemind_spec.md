# 🧠 HiveMind Protocol — Спецификация

## Четырёхуровневая модель

| Уровень | Название | Функция | Метафора | Пример команды |
|:---|:---|:---|:---|:---|
| **IV** | Дух / Проклятие | Верховный этический запрет | Судья, Совесть, Конституция | "Не навреди", "Сохрани жизнь" |
| **III** | Приказ / Воля | Стратегия и принятие решений | Генерал, Совет, CEO, Монарх | "Найти воду", "Построить базу" |
| **II** | Право / Контракт | Тактика и распределение | Юрист, Аукционист, KPI | "Тендер на бурение", "Приказ отделению" |
| **I** | Сеть / Дело | Исполнение и связь (DTN) | Солдат, Рабочий, Ровер | "Бурить здесь", "Доставить груз" |

## Модели взаимодействия

### Военная модель
Приказ (III) -> Исполнение (I) -> Отчёт (I -> III)
- Применение: критические миссии, аварии, военные операции.
- Особенность: нет обсуждения, приказ не оспаривается.

### Корпоративная модель
Тендер (II) -> Ставка (I) -> Контракт (II) -> Исполнение (I) -> Прибыль
- Применение: добыча ресурсов, торговля, логистика.
- Особенность: аукцион CBBA, оптимизация прибыли.

### Научная модель
Гипотеза (III) -> Данные (I) -> Обсуждение (II) -> Консенсус (III)
- Применение: геологоразведка, исследования, поиск жизни.
- Особенность: решение через доказательства, а не приказ.

### Демократическая модель
Предложение (II) -> Голосование (I -> II) -> Решение большинства (III) -> Исполнение (I)
- Применение: управление колонией, распределение общих ресурсов.
- Особенность: медленно, но устойчиво к ошибкам одного лидера.

### Монархическая модель (Сакральная вертикаль)
Благословение (IV -> III) -> Завет (III <-> I) -> Правление (III -> I) -> Отчёт (I -> III) -> Подтверждение Завета (I -> II)
- Применение: колонии, основанные на религиозной или традиционной миссии; долгосрочные династические проекты.
- Особенность: власть Монарха исходит от Духа (Слой IV), а не от приказа или голосования. Народ подтверждает Завет через признание (Слой II). Монарх несёт личную ответственность перед Богом за каждый приказ. Смещение невозможно без нарушения сакрального контракта.

## Роли в HiveMind

| Роль | Уровень | Описание |
|:---|:---|:---|
| Супервизор Роя | III (Воля) | Принимает стратегические решения за рой |
| Агент-специалист | I-II (Сеть-Право) | Исполняет задачи своей профессии |
| Теолог / Этик | IV (Дух) | Проверяет решения на соответствие миссии |
| Аукционист | II (Право) | Ведёт торги за ресурсы и задачи |
| Монарх / Хранитель Завета | III-IV (Воля-Дух) | Правит по благословению, отчитывается перед Богом |

## Протокол связи

- **Формат**: JSON строгой схемы
- **Транспорт**: DTN (Delay/Disruption Tolerant Networking)
- **Эмуляция задержек**: Земля-Луна 2.5 сек, Земля-Марс 5-20 мин
- **Экономика**: запросы оплачиваются токенами FFF (защита от спама)
