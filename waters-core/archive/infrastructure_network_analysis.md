# 🌐 Инфраструктура и сетевая архитектура WATERS

**Статус:** Черновик Системного Архитектора. Фаза 0.
**Дата:** 05.05.2026

---

## 1. Операционные системы

| Критерий | Ubuntu 26.04 LTS | osModa / NixOS | Kortix |
|:---|:---|:---|:---|
| **Суть** | Классический серверный Linux | AI-Native ОС | ОС для AI-компании |
| **AI-интеграция** | Локальные AI-инструменты | 91 AI tool, 10 демонов | 60+ навыков, 3000+ интеграций |
| **Безопасность** | Стандартные практики | Атомарный откат, квантовая защита | Доступ к ФС, нужна изоляция |
| **Сложность** | Низкая | Высокая | Средняя |

### Выбор:
- **Старт:** Ubuntu 26.04 LTS
- **Продакшен:** osModa/NixOS (атомарные откаты, аудит)

---

## 2. Сетевая архитектура

### Гибридная модель:

1. **Центральное ядро (VPS):**
   - Оркестратор, RAG, Kafka/Redis, тяжёлые модели
   - Ubuntu Server + Docker + OpenCode Server

2. **Локальный полигон (ноутбук):**
   - Разработка, отладка
   - VS Code + OpenCode CLI

3. **Автономные агенты ("пирожки"):**
   - Легковесные Docker-контейнеры
   - qwen2.5-coder:7b + мини-RAG

---

## 3. Брокеры сообщений

| Критерий | Redpanda | Apache Kafka | NATS JetStream |
|:---|:---|:---|:---|
| **Совместимость с Kafka API** | Полная | Полная | Через адаптер |
| **Лицензия** | Apache 2.0 | Apache 2.0 | Apache 2.0 |
| **Потребление RAM** | Низкое | Высокое | Очень низкое |
| **Сложность** | Низкая (один бинарник) | Высокая (ZooKeeper) | Средняя |
| **Retention** | Да | Да | Ограничено |

### Выбор:
- **Старт:** Redpanda (Kafka-совместима, легче)
- **Рост:** Kafka (когда агентов > 10 000)
