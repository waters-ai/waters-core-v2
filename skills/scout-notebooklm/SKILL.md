## Назначение

Валидация и экстракция содержимого через Google NotebookLM. Загружает найденные документы (PDF, JSON, транскрипции), проверяет их качество и извлекает выжимку.

## Входные данные

| Поле | Тип | Описание |
|------|-----|----------|
| file_path | string | Путь к файлу для анализа |
| query | string | Исходный поисковый запрос (для проверки релевантности) |

## Выходные данные (на файл)

| Поле | Тип | Описание |
|------|-----|----------|
| verdict | string | `годно` / `мусор` |
| summary | string | Краткая выжимка (3-5 предложений) |
| is_valid_source | bool | NotebookLM считает источник авторитетным |
| is_relevant | bool | Релевантно поисковому запросу |

## Выходные данные (на задачу — синтез)

| Поле | Тип | Описание |
|------|-----|----------|
| summary | string | Общая сводка по всем валидным файлам |
| contradictions | string | Противоречия между источниками (если есть) |

## Алгоритм

### Per-file:
1. Создать Notebook с именем файла
2. Загрузить файл как источник
3. Запросить: `годно/мусор` (оценка источника)
4. Запросить: `да/нет` (релевантность запросу)
5. Запросить: выжимка (3-5 предложений)
6. Если `годно` + `да` → verdict годно
7. Удалить Notebook

### Task synthesis:
1. Создать Notebook со всеми годными файлами
2. Запросить: общая сводка + противоречия
3. Удалить Notebook

## Бесплатно

NotebookLM не требует API-ключа. Использует Google-аккаунт через notebooklm-py (0 токенов).
