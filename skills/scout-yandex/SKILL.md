## Назначение

RU-fallback поисковый движок через Yandex.XML. Используется когда DuckDuckGo недоступен из RU-зоны.

## Входные данные

| Поле | Тип | Описание |
|------|-----|----------|
| query | string | Поисковый запрос |
| max_results | int | Макс. результатов (default: 10) |

## Выходные данные

| Поле | Тип | Описание |
|------|-----|----------|
| title | string | Заголовок |
| url | string | Ссылка |
| snippet | string | Описание |

## Требования

- `YANDEX_API_KEY` — API-ключ Yandex.XML
- `YANDEX_USER` — логин Yandex.XML
- Регистрация на https://xml.yandex.ru/ (бесплатно, 10000 запросов/день)

## Статус

По умолчанию неактивен. Активируется при наличии ключей.
Без ключей Scout работает только через DuckDuckGo + YouTube.
