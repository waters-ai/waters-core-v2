# 🤝 Как внести вклад в WATERS

## Принципы

- Научная обоснованность превыше всего
- Код должен быть совместим со Space ROS / SpaceMind
- Все агенты проходят проверку через HiveMind (этический слой)
- Мы ценим открытость и конструктивную критику

## Процесс

### 1. Нашли идею?
Создайте **Issue** с тегом `proposal`. Опишите:
- Что предлагаете
- Почему это важно
- Какие есть альтернативы

### 2. Хотите написать код?
1. Форкните репозиторий
2. Создайте ветку: `feature/имя-вашей-фичи`
3. Напишите код, следуя нашим стандартам
4. Проверьте, что тесты проходят
5. Откройте **Pull Request**

### 3. Ваш PR проверят
- **Mythos Agent** (автоматическая проверка безопасности)
- **AI-валидатор** (соответствие доктрине)
- **Рецензенты-люди** (экспертная оценка)

### 4. Ваш вклад принят!
Вы становитесь **Контрибьютором WATERS**. Ваше имя — в истории проекта.

## Стандарты кода

- Python 3.10+ для оркестратора
- TypeScript для интерфейсов
- Docker для песочниц
- JSON-схемы для всех API-контрактов

## Вопросы?

Создайте Issue с тегом `question` или пишите в обсуждения.
